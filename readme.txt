How to use:
Open AnotherChip8.exe, choose file to emulate (.ch8 extension), press esc to kill the app.
In case of window close without a reason, open log.txt and check out the problem.


Chip8 keypad:

=========
|1|2|3|C|
|4|5|6|D|
|7|8|9|E|
|A|0|B|F|
=========


PC Keyboard setting:

=========
|1|2|3|4|
|q|w|e|r|
|a|s|d|f|
|z|x|c|v|
=========
